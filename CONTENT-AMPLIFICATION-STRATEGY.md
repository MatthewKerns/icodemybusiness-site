## 🎯 The Core Loop

1. **BUILD** a feature (30min - 2hrs)
2. **DOCUMENT** in blog post (15-30min using template)
3. **AMPLIFY** across 4 platforms (30min total)
4. **CONVERT** to discovery calls → paid consulting

---

## 📝 Blog Post Creation (15-30 min)

Use the template at `/blog/post-template.html`

### Quick Formula:
- **Problem** (1-2 sentences) - What sucked?
- **Solution** (2-3 sentences) - What you built
- **Key Insight** (1 trick + 5-10 lines of code)
- **Results** (3 bullet points)
- **Next** (What's coming)

### Time Savers:
- Don't over-document
- Screenshots > long explanations
- One key code snippet > full implementation
- Results > process

## 🚀 Platform Amplification Strategy

### 1. X (Twitter) - The Technical Crowd
**Format:** Thread (5-7 tweets)
```
Tweet 1: Hook + Problem
"I was spending 2hrs/day on [TASK]. Built an AI agent to handle it. Now it's 0.

Here's exactly how: 🧵"

Tweet 2: What you built (1 sentence + screenshot)
Tweet 3: The key insight/trick
Tweet 4: Results (bullets)
Tweet 5: Link to blog post
Tweet 6: CTA for discovery call
Tweet 7: "Follow for more AI agent builds"
```

**Hashtags:** #buildinpublic #AI #automation #nocode #[wellness OR business]

---

### 2. TikTok - The Visual Story
**Format:** 60-second screen recording

**Script Template:**
```
0-3s: "POV: You're wasting [X hours] on [PROBLEM]"
3-10s: Show the problem visually
10-15s: "So I built this..." (show solution running)
15-45s: Quick demo of it working
45-55s: Show results/time saved
55-60s: "Link in bio for the blueprint"
```

**Tips:**
- Use CapCut for quick edits
- Add captions (85% watch without sound)
- Trending audio when relevant
- End with clear CTA

---

### 3. YouTube - The Deep Dive
**Format:** 5-10 minute implementation video

**Structure:**
1. **Hook** (0-15s): Problem + promise
2. **Demo** (15s-1min): Show it working
3. **Build** (1-5min): Key implementation steps
4. **Results** (5-7min): Metrics + benefits
5. **CTA** (7-8min): Discovery call + next video

**Thumbnail Formula:**
- Split screen: Messy before → Clean after
- Big text: "AI Agent Saved Me 10hrs/Week"
- Your face with surprised/satisfied expression

---

### 4. Instagram - The Lifestyle Angle
**Format:** Carousel (5-7 slides) + Reel

**Carousel Structure:**
```
Slide 1: Hook image + problem statement
Slide 2: "What if you never had to..."
Slide 3: Solution overview (visual)
Slide 4: Key benefit #1
Slide 5: Key benefit #2
Slide 6: Results screenshot
Slide 7: CTA - "Link in bio for free discovery call"
```

**Reel:** Repurpose TikTok content
